DIY Photoshop in p5.js

This is a space themed drawing application, coded in p5.js.

Controls:
1–4: hard brushes
5: airbrush
6–7: star brushes
8–0: animated stamps
'e': eraser
'x': delete stamp tool


'/': set background color
'.': clear canvas
'z': undo
'a': toggle blend mode (screen)
'p': save image
